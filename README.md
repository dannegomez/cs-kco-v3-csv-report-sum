# cs-kco-v3-csv-report-sum
Simple c# program to summerize custom reports from Klarna checkout v3.

Sums FEE, SALE and RETURN in a datagridview.

# How to use
- Export custom report from KCO
- Select exported file
- Select currency
- Click on summerize
